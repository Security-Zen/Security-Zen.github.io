Quick Start

Below is a quick introduction to the Security Zen top level menu items and what they do. Select buttons below for additional details.

Connections

Connections enable you to connect to your Salesforce environments such as Sandboxes and Production. Establishing a connection is the first step to analyzing security.

Configure

Once a connection is established, select Configure->Configure Security to choose which security files (Profiles and/or Permission Sets) and which entities (Objects, Fields, Record Types, etc) you want to retrieve. You must select at least one security file and at least one entity in order to retrieve security.

You can also create ANT package files using Configure->Create ANT Package File.
