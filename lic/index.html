License

License Options:

  • Free: Just download Security Zen from our Download page, install it, and try it out! You get most of the functionality and no time limits.

  • Security Zen Master - 1 Year Term Node Locked License: $49.95 per year. Unlocks the full potential of the software including the ability to deploy security changes to other environments! License will be issued for a particular machine based on the Unique Identifier (UID) you provide at the time of purchase and is valid for one year.

  • Security Zen Master - Perpetual Node Locked License: $199.95. Unlocks the full potential of the software including the ability to deploy security changes to other environments! License will be issued for a particular machine based on the Unique Identifier (UID) you provide at the time of purchase and never expires.

Additional Features of Security Zen Master Licenses:

  • Ability to deploy your security changes!
  • No hard limits on number of Profiles or Permission Sets that can be retrieved. The Free version is limited to 5 security files per retrieve.
  • Ability to retrieve Security Files and Entities based on previously retrieved workspace.
  • Ability to retrieve Profiles and Permissions Sets by User.
  • Ability to retrieve Permission Sets by Permission Set Group.
  • Ability to filter the selection of entities to be retrieved.
  • Display Profiles and Permission Sets (and security granted by them) by selecting a User.
  • Display the list of Users with a selected Profile or Permission Set.
  • Ability to import Metadata files for use with ANT deployments.
  • Ability to export Metadata files for use with ANT deployments.
  • Ability to perform Security Assignment Analysis.
  • The warm fuzzy feeling that you are contributing to the development of an awesome tool!
